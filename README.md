# 御見積書ジェネレーター

ホシザキ中国株式会社 サービス担当者向け、ブラウザで動作する御見積書作成ツールです。
PDF・Excel の両形式で出力でき、インターネット接続不要（初回ロード後）で現場でも使用できます。

---

## バージョン

| バージョン | ディレクトリ | 状態 |
|-----------|------------|------|
| V2        | `v2/`      | 安定版（参照用） |
| V3        | `v3/`      | 最新版（推奨） |

## V3 の使い方

`v3/index.html` をブラウザで開くだけで動作します。サーバー不要です。

### 基本的な流れ

1. **基本情報タブ** — 顧客名・機器情報・故障原因を入力
2. **部品タブ** — 使用部品を追加（仕入・売価・粗利を自動計算）
3. **作業費タブ** — 作業費・値引きを入力
4. **設定・出力タブ** — 担当者選択 → PDF または Excel で出力

各タブ下部の「次へ →」ボタンで順番に進むことができます。

## V3 新機能（V2 からの改善点）

### 1. 部品カード折りたたみ
- 部品カードがデフォルトで折りたたみ表示になり、スマホでのスクロール量を大幅に削減
- 折りたたみ時は「部品名・数量・売価」のみ表示。`▼ 詳細` をタップして全フィールドを展開

### 2. 発行日の自動設定
- 新規作成時に発行日が自動で「今日の日付」にセットされます
- V2 では日付がハードコードされていたため毎回手動変更が必要でしたが、V3 では不要

### 3. タブ間ナビゲーション（次へ/前へ）
- 各タブ最下部に「← 前へ」「次へ →」ボタンを追加
- ボタン押下時は自動的にページ上部へスクロールするため、次のタブを探す手間がなくなります

### 4. 作業費クイック追加ボタン
- 作業費タブにクイック追加ボタンを設置（基本出張費・技術料・調整費・薬品洗浄作業費・夜間割増・日祭日割増）
- ボタンタップで項目名を自動セットし、金額フィールドへ自動フォーカス

### 5. 担当者リストの永続化
- 担当者リストが専用ストレージキー（`mitsumori_v2_staff`）で独立管理されるように改善
- V2 ではブラウザキャッシュ消去で担当者が消えていましたが、V3 では安定して保持
- V2 のデータが残っている場合は自動的に移行します
- ソースコードから氏名を除去（個人情報対策）

### 6. 故障原因・処置の定型文挿入
- 「製氷不良・冷却不良・漏水・異音・電気系統不良・経年劣化」の定型文ボタンを設置
- テキストエリアが空の場合は挿入、入力済みの場合は末尾に追記

### 7. PDF・Excel の日付フォーマット統一
- 発行日・設置日の表示を `YYYY年MM月DD日` 形式に統一（V2 では Excel のみ `YYYY-MM-DD` 形式で不一致）
- Excel の金額セルに `#,##0` フォーマットを明示適用（カンマ区切りで見やすく）

### その他の改善
- **リセットボタンを出力ボタン群から分離**: 誤タップによるデータ消去を防止
- **担当者未登録時のガイド表示**: 初回起動時に担当者登録を促すトーストを表示

## データの保存について

入力データはブラウザの `localStorage` に自動保存されます。

| キー | 内容 |
|-----|------|
| `mitsumori_v2_data` | フォーム入力・部品・作業費データ |
| `mitsumori_v2_templates` | 保存済みテンプレート |
| `mitsumori_v2_staff` | 担当者リスト（V3 から独立管理） |
| `mitsumori_theme` | テーマ設定（ライト/ダーク） |

> ⚠️ ブラウザのキャッシュ・サイトデータを消去するとデータが失われます。
> 重要なデータはテンプレートとして保存するか、PDF/Excel で出力してください。
